В качестве реализации поиска по json используется библиотека https://github.com/json-path/JsonPath

В данной библиотеке определен язык поиска

Ниже представлены несколько примеров

  • $ - вывести полный JSON
  • $.products - вывести список продуктов
  • $.products[1] - получить 2й продукт в списке
  • $.products[1].A2.description - получить описание 2го продукта в списке
  • $.products[*].*.description - получить описания всхе продуктов
